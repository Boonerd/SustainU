This is a sustainability-focused web application that empowers communities to take action on climate change through education, data visualization, and community engagement.

## Overview

SustainU is a platform designed to raise awareness about environmental issues and provide actionable insights for sustainable living. The project features multiple sections including personal stories, data visualization, and practical sustainability tips. It is for the purpose of my course project but I am open to ideas on how to make it better. 

## Features

- **Dynamic Hero Section** - Engaging landing page with clear call-to-action
- **The Voices** - Personal stories and testimonials from environmental advocates
- **The Data** - Interactive charts and climate data visualization
- **Act Now** - Practical sustainability tips and community action tracker
- **Dark/Light Mode** - User-friendly theme switching for comfortable viewing
- **Responsive Design** - Works seamlessly across desktop, tablet, and mobile devices

## Tech Stack

- **React** - Frontend framework
- **React Router** - Navigation and routing
- **Tailwind CSS** - Styling and responsive design
- **Lucide React** - Icon library
- **Vite** - Build tool and dev server

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Boonerd/SustainU
cd sustainu
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

## Color Palette

The project uses an eco-inspired color scheme:

**Light Mode:**
- Background: `#faf9f6` (Soft cream)
- Surface: `#ffffff` (White)
- Primary: `#2d6a4f` (Forest green)
- Accent: `#f4a261` (Warm orange)

**Dark Mode:**
- Background: `#0f1419` (Deep blue-gray)
- Surface: `#1a1f2e` (Slate)
- Primary: `#7fcd91` (Soft green)
- Accent: `#f4a261` (Warm orange)

## Usage

The application is designed to be intuitive:

1. Navigate through sections using the top navigation bar
2. Toggle between light and dark modes using the sun/moon icon
3. Click "Start Your Journey" to begin exploring sustainability tips
4. Browse through stories in "The Voices" section
5. Analyze climate data in "The Data" section
